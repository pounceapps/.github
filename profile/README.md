## The apps

### PouncePad
A simple, **AI-drivable text editor** — a TextEdit/Notepad replacement with syntax
highlighting for ~30 languages, Markdown + Mermaid preview, crash-safe tabs, and a
**Claude Code channel** built in: type in the app, Claude answers right there.

</td>
</tr>
<tr>
<td width="90" align="center"><img src="https://raw.githubusercontent.com/pounceapps/.github/main/profile/assets/yarn.png" width="64" alt="PounceSQL"></td>
<td>

### PounceSQL
A **native SQL client** for Azure SQL, SQL Server, PostgreSQL & SQLite — with an AI
assistant, rich results (charts, diagrams, KPIs), and a built-in MCP server so Claude
can drive the whole app.

</td>
</tr>

## Install

```sh
brew tap pounceapps/tap
brew install --cask pouncesql   # the SQL client
brew install --cask pouncepad   # the editor
```

Prefer a direct download? Grab the notarized DMGs from **[pounceapps/downloads](https://github.com/pounceapps/downloads)**.
