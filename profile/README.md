# PounceApps 🐾

Small, fast, **native macOS apps** with a retro heart and AI built in.

## The apps

| App | What it is |
| --- | --- |
| **PounceSQL** | Native SQL client for Azure SQL, SQL Server, PostgreSQL & SQLite — with an AI assistant and a built-in MCP server. |
| **PouncePad** | A simple, AI-drivable text editor — with a Claude Code channel built right in. |

## Install

```sh
brew tap pounceapps/tap
brew install --cask pouncesql
brew install --cask pouncepad
```
